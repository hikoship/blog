Games Reviews

FIFA 98
1998 · Windows · ✭✭✭✭✭
我的第一款游戏，可以用鼠标操作，和小学同学在家里合作打法国世界杯。

Need for Speed 3 / 极品飞车 3
1998 · Windows · ✭✭✭
赛车游戏启蒙，但对于当时的我来说太难了，有挫败感。

Red Alert / 红色警戒
2000? · Windows · ✭✭✭
即时战略启蒙，被电脑虐惨了。

Red Alert 2 / 红色警戒 2
2004? · Windows · ✭✭✭✭
即时战略启蒙，被电脑虐惨了。

Road Rash / 暴力摩托
2004 · Windows · ✭✭✭✭✭
趣味十足，百玩不厌的赛车小游戏。

FIFA 04
2004 · Windows · ✭✭✭
只记得我当年一球未进。

NBA Live 03
2004 · Windows · ✭✭✭
只记得可以自动空中接力。

NBA Live 04
2004 · Windows · ✭✭✭✭
第一个长时间玩的 NBA 游戏。使用美国明星队，体会扣篮、三分的快感。

三国群英传 2
2004 · Windows · ✭✭✭
第一个三国群英传游戏。

三国群英传 3
2004 · Windows · ✭✭✭✭
长时间玩的三国群英传游戏。画面大幅提升，增加很多必杀技和武将技以及道具。但武将不能培养。

口袋妖怪：叶绿
2004 · GBA · ✭✭✭
第一个口袋妖怪游戏，实现了操控神奇宝贝的梦想。但对系统完全不了解，用爱在玩。

KOF 2002
2004 · Neo Geo · ✭✭✭✭✭
至今仍然偶尔把玩。系统平衡，人物丰富，打好了快感十足。

三国志曹操传
2004 · Windows · ✭✭✭✭✭
精美的战棋小品，也可以说是我的战棋启蒙，玩起来欲罢不能。系统玄妙，MOD 众多，至今吸引众人研究。

三国群英传 4
2005 · Windows · ✭✭✭✭
其实是饱受诟病的一代，取消了城内地图，军师计的作用也很奇怪。但回归了即时策略，对战也更加宏大爽快。打神仙、加基础值让平板武将也有春天。日本势力也很有意思。

QQ 堂
2005 · Windows · ✭✭✭
我的第一款网游。清新可爱，模式新颖，抄自泡泡堂是黑点，但其实更加有趣。后期外挂泛滥，几乎不能玩。

PES 4 / 实况足球 8
2005 · Windows · ✭✭✭✭
玩的不多，在浩方上对战过几盘，节奏快。

PES 5 / 实况足球 9
2005 · Windows · ✭✭✭✭
第一款实况游戏，并不怎么得其要领，但充满了和初中同学对战的回忆。

NBA Live 06
2005 · Windows · ✭✭✭✭
很有趣的一代 NBA 游戏，全明星周末精彩纷呈，王朝模式可以组建无敌之师，全明星球员能做出各种精彩的动作。

FIFA 05
2005? · Windows · ✭✭✭✭
虽然还是不得要领，但玩起来比较开心。还有第三客场球衣，很有意思。

街头篮球
2006 · Windows · ✭
篮球网游，对网速和队友要求太高，而且经常组不到队友。

三国群英传 6
2006 · Windows · ✭
几乎没有创新的游戏。取消了日本势力，各种妖怪肆虐。

三国志 11
2006 · Windows · ✭✭✭
极其精美的水墨风格画面，可惜不得要领，没怎么多玩。

真三国无双 4：特别版
2006 · Windows · ✭✭✭✭
我的第一个款无双游戏。体会了无敌的割草快感。武将各有特色，关卡设计也比较有趣。

跑跑卡丁车
2006 · Windows · ✭✭✭✭
非常好玩的一款网游，在新手训练中引擎发动的那一刻我就发现自己爱上了这种感觉。道具赛充满趣味，竞速赛又有一定的驾驶感。后期道具太贵了。

实况足球 10
2006 · Windows · ✭✭✭✭✭
实况系列集大成的巅峰之作。节奏，打法和爽快感都达到了平衡，几乎没什么缺点。还有各种鸵鸟恐龙球衣，可惜后面都不见了。

NBA Live 07
2006 · Windows · ✭✭
没什么新意的续作，只能用摇杆搓出球星招式，新秀数值奇怪。

口袋妖怪：红宝石
2007? · GBA · ✭✭✭✭
第三世代初体验，奠定了诸多新系统，新宠强宠众多，剧情精彩。

超级机器人大战 A
2007? · GBA · ✭✭✭
我的第一款机战，但陆陆续续花了五年才通关，感谢汉化组。剧情是玩了 OG2 之后才了解的，很喜欢。夺魂者是我最爱的机体。可惜画面机能当时还未被榨干，而且不能关动画。

PES 2008
2007 · Windows · ✭✭
PS3 时代实况初作，跑步像滑冰，速度型球员的天下。

NBA Live 08
2007 · Windows · ✭✭
依然毫无新意。最终成为了 NBA Live 的末代作品。

口袋妖怪：绿宝石
2008? · GBA · ✭✭✭✭✭
GBA 上最经典的作品之一，也是口袋妖怪系列中数得上的优秀版本。集红蓝宝石剧情之大成，神兽决战气势磅礴，选美大赛自成一派，战斗边疆别有洞天。
